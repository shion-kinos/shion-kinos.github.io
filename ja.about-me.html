自己紹介

こんにちは。木下至遠（きのした・しおん）です。

趣味など

言語オタクです。言語と言語学が大好きです。英語と日本語が流暢で、フランス語が中級レベルです。広東語と漢文を始めとする中国の諸言語にも興味があります。国立国会図書館のデジタルアーカイブにのめり込んで今の辞書には載っていない古い言葉を発見したり、古代中国の哲学書を解読してその不朽の知恵を吸収しようとしたりするのも楽しいです。オンライン辞書に読み耽ることもあります。言語は本当に限りなく面白いです。

趣味でコーディングもやっています。以前は Python で色々書いていましたが、最近は主に HTML と JavaScript を使ってウェブサイトや自分用のツールを作っています。下手すると何時間もコーディングに没頭して寝食を忘れてしまいます。

世界観設定とサイエンスフィクションも好きです。最近は SF 物語を書き始めて、今はこれが一番の趣味となっています。

困っている人を助けることにすごく熱心で、恵まれない環境にある人々の支援に尽力する地元団体でよくボランティア活動に参加しています。

友人の感想

「至遠くん、君は博学でありながらふざけることも大好きなやつなんだな」

【至遠】高校時代のある日、化学の先生に「屋外便所に行っていいですか？」って聞いたら「至遠くん……便所はとっくの昔に屋内に移されています」って言われた
【友人】至遠くん、君は博学でありながらふざけることも大好きなやつなんだな

そう、そういう者に私はなりたい。
